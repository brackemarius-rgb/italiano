# Verlaufsform: stare + gerundio

Drückt aus, dass eine Handlung **gerade in diesem Moment** stattfindet (engl. *I am doing*).

## 1. Aufbau

```
stare (konjugiert) + gerundio
```

## 2. stare im Präsens

|             |        |
| ----------- | ------ |
| io          | sto    |
| tu          | stai   |
| lui/lei     | sta    |
| noi         | stiamo |
| voi         | state  |
| loro        | stanno |

## 3. Gerundio – Bildung

| Infinitiv-Endung | Gerundio-Endung | Beispiel               |
| ---------------- | --------------- | ---------------------- |
| -are             | -ando           | parlare → parl**ando** |
| -ere             | -endo           | prendere → prend**endo** |
| -ire             | -endo           | dormire → dorm**endo** |

**Unregelmäßig:**
- *fare → facendo*
- *dire → dicendo*
- *bere → bevendo*

## 4. Beispiele

- *Sto mangiando.* — Ich esse gerade.
- *Cosa stai facendo?* — Was machst du gerade?
- *Marco sta dormendo.* — Marco schläft gerade.
- *Stiamo parlando di te.* — Wir reden gerade über dich.

## 5. Wichtig: Wann **nicht** verwenden?

Anders als im Englischen wird die Verlaufsform **nicht** für Pläne oder Zukünftiges benutzt:
- ❌ *Sto andando a Roma domani.*
- ✓ *Domani vado a Roma.* / *Domani andrò a Roma.*

Auch im Imperfetto möglich (= "war dabei zu tun"):
- *Stavo dormendo quando ha chiamato.* — Ich schlief gerade, als er anrief.

## 6. Pronomen

Pronomen können **vor stare** oder **am Gerundio** angehängt werden:
- *Lo sto leggendo.* = *Sto leggendolo.*
- *Mi sto vestendo.* = *Sto vestendomi.*

## 7. Häufige Fehler

- *Sono mangiando* ❌ → *Sto mangiando* ✓ (immer stare!)
- *Sto mangia* ❌ → *Sto mangiando* ✓
- *Sto facciendo* ❌ → *Sto facendo* ✓
