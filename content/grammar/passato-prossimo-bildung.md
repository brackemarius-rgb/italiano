# Passato prossimo — Bildung

Das **passato prossimo** ist die wichtigste Vergangenheitsform für **abgeschlossene Handlungen**. Es entspricht in der gesprochenen Sprache dem deutschen Perfekt.

## 1. Aufbau

```
passato prossimo = Präsens von essere/avere + Partizip Perfekt
```

Beispiele:
- *Ho mangiato la pizza.* — Ich habe Pizza gegessen.
- *Sono andato al cinema.* — Ich bin ins Kino gegangen.

## 2. Partizip Perfekt – regelmäßig

| Infinitiv-Endung | Partizip-Endung | Beispiel                |
| ---------------- | --------------- | ----------------------- |
| -are             | -ato            | parlare → parl**ato**   |
| -ere             | -uto            | credere → cred**uto**   |
| -ire             | -ito            | dormire → dorm**ito**   |

## 3. Wichtige unregelmäßige Partizipien

| Infinitiv | Partizip   |   | Infinitiv | Partizip   |
| --------- | ---------- | - | --------- | ---------- |
| essere    | stato      |   | aprire    | aperto     |
| fare      | fatto      |   | offrire   | offerto    |
| dire      | detto      |   | scrivere  | scritto    |
| venire    | venuto     |   | leggere   | letto      |
| rimanere  | rimasto    |   | vedere    | visto      |
| nascere   | nato       |   | prendere  | preso      |
| morire    | morto      |   | mettere   | messo      |
| chiedere  | chiesto    |   | chiudere  | chiuso     |
| rispondere| risposto   |   | bere      | bevuto     |
| vivere    | vissuto    |   | scegliere | scelto     |

## 4. Mit avere — Beispiele

- *Ho parlato con Marco.* — Ich habe mit Marco gesprochen.
- *Hai mangiato bene?* — Hast du gut gegessen?
- *Abbiamo visto un film.* — Wir haben einen Film gesehen.
- *Hanno comprato una casa.* — Sie haben ein Haus gekauft.

## 5. Mit essere — Anpassung an Subjekt

Bei **essere** richtet sich das Partizip nach Genus und Numerus des Subjekts:

|             | maskulin             | feminin               |
| ----------- | -------------------- | --------------------- |
| Singular    | sono andat**o**      | sono andat**a**       |
| Plural      | siamo andat**i**     | siamo andat**e**      |

- *Marco è andat**o**.* — Marco ist gegangen.
- *Maria è andat**a**.* — Maria ist gegangen.
- *I ragazzi sono partit**i**.* — Die Jungen sind abgefahren.
- *Le ragazze sono partit**e**.* — Die Mädchen sind abgefahren.

Welche Verben mit *essere* gehen, siehe → [passato-prossimo-essere-vs-avere](passato-prossimo-essere-vs-avere.md).

## 6. Häufige Fehler

- *Mangiato la pizza* ❌ → *Ho mangiato la pizza.* ✓ (Hilfsverb darf nicht fehlen)
- *Ho prendato* ❌ → *Ho preso* ✓
- *Maria è andato* ❌ → *Maria è andata* ✓
- *Loro hanno andato* ❌ → *Loro sono andati* ✓ (andare braucht essere)
