# Condizionale (Konditional)

Der Conditional drückt **Höflichkeit, Wünsche, Ratschläge und hypothetische Aussagen** aus.

## 1. Bildung – regelmäßig

Stamm wie beim Futuro + Endungen **-ei, -esti, -ebbe, -emmo, -este, -ebbero**:

|             | parlare       | prendere       | dormire       |
| ----------- | ------------- | -------------- | ------------- |
| io          | parlerei      | prenderei      | dormirei      |
| tu          | parleresti    | prenderesti    | dormiresti    |
| lui/lei     | parlerebbe    | prenderebbe    | dormirebbe    |
| noi         | parleremmo    | prenderemmo    | dormiremmo    |
| voi         | parlereste    | prendereste    | dormireste    |
| loro        | parlerebbero  | prenderebbero  | dormirebbero  |

## 2. Unregelmäßig

Gleicher Stamm wie beim Futuro:

|             | essere     | avere     | volere     | potere     |
| ----------- | ---------- | --------- | ---------- | ---------- |
| io          | sarei      | avrei     | vorrei     | potrei     |
| tu          | saresti    | avresti   | vorresti   | potresti   |
| lui/lei     | sarebbe    | avrebbe   | vorrebbe   | potrebbe   |
| noi         | saremmo    | avremmo   | vorremmo   | potremmo   |
| voi         | sareste    | avreste   | vorreste   | potreste   |
| loro        | sarebbero  | avrebbero | vorrebbero | potrebbero |

## 3. Verwendung

### a) Höflicher Wunsch / höfliche Bitte

- *Vorrei un caffè, per favore.* — Ich hätte gern einen Kaffee.
- *Potresti aiutarmi?* — Könntest du mir helfen?
- *Mi darebbe un'informazione?* — Würden Sie mir eine Auskunft geben?

### b) Ratschlag

- *Dovresti studiare di più.* — Du solltest mehr lernen.
- *Faresti meglio a riposare.* — Du würdest besser ruhen.

### c) Wunsch / Traum

- *Mi piacerebbe andare in Italia.* — Ich würde gern nach Italien fahren.
- *Comprerei una casa al mare.* — Ich würde ein Haus am Meer kaufen.

### d) Hypothese

- *Sarebbe bello.* — Das wäre schön.
- *Al posto tuo, parlerei con lui.* — An deiner Stelle würde ich mit ihm reden.

## 4. Wichtige Höflichkeitsfloskeln

- *vorrei* — ich hätte gern
- *potrei?* — könnte ich?
- *potresti?* — könntest du?
- *mi piacerebbe* — ich würde gern
- *dovrei* — ich sollte
- *avrei voglia di* — ich hätte Lust auf

## 5. Häufige Fehler

- *Voglio un caffè* (im Restaurant) → klingt unhöflich → besser *Vorrei un caffè* ✓
- *parlerei* in 1. Plural ❌ → *parleremmo* (mit Doppel-m!) ✓
- *vorebbe* ❌ → *vorrebbe* (Doppel-r) ✓
