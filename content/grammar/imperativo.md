# Imperativo (Befehlsform)

Der Imperativ wird verwendet für **Befehle, Aufforderungen, Anweisungen, Ratschläge und Anweisungen** (z. B. in Rezepten, Wegbeschreibungen).

## 1. Formen – tu / Lei / noi / voi

| Person | parlare    | prendere    | dormire    | finire (-isc-) |
| ------ | ---------- | ----------- | ---------- | -------------- |
| (tu)   | parla!     | prendi!     | dormi!     | finisci!       |
| (Lei)  | parli!     | prenda!     | dorma!     | finisca!       |
| (noi)  | parliamo!  | prendiamo!  | dormiamo!  | finiamo!       |
| (voi)  | parlate!   | prendete!   | dormite!   | finite!        |

**Wichtig:** Bei -are-Verben enden tu auf **-a**, Lei auf **-i** (es ist umgekehrt zur 3. Person!).
Bei -ere/-ire-Verben enden tu auf **-i**, Lei auf **-a**.

## 2. Wichtige unregelmäßige Imperative (tu)

| Verb    | tu      | (Lei)   | Beispiel                |
| ------- | ------- | ------- | ----------------------- |
| essere  | sii     | sia     | *Sii buono!*            |
| avere   | abbi    | abbia   | *Abbi pazienza!*        |
| andare  | va' / vai | vada  | *Va' a casa!*           |
| dare    | da' / dai | dia   | *Dammi un bacio!*       |
| fare    | fa' / fai | faccia| *Fa' presto!*           |
| dire    | di'     | dica    | *Dimmi tutto!*          |
| stare   | sta' / stai | stia| *Sta' zitto!*           |
| sapere  | sappi   | sappia  | *Sappi che…*            |

## 3. Verneinter Imperativ

### tu-Form: NON + Infinitiv

- *Non parlare!* — Sprich nicht!
- *Non andare!* — Geh nicht!
- *Non avere paura!* — Hab keine Angst!

### Lei/noi/voi: NON + normaler Imperativ

- *Non parli!* (Lei)
- *Non parliamo!* (noi)
- *Non parlate!* (voi)

## 4. Pronomen am Imperativ

### tu/noi/voi: Pronomen werden **angehängt**

- *Parlami!* — Sprich zu mir!
- *Aspettatela!* — Wartet auf sie!
- *Andiamoci!* — Gehen wir hin!
- *Diamogli un libro!* — Geben wir ihm ein Buch!

**Verdoppelung** bei einsilbigen Verben:
- *dare → dammi* (gib mir!)
- *fare → fammi* (mach mir!)
- *dire → dimmi* (sag mir!)
- *stare → stammi* (bleib mir!)
- *andare → vacci* (geh hin!)

### Lei: Pronomen wird **vorgestellt**

- *Mi dia un libro!* — Geben Sie mir ein Buch!
- *Lo prenda!* — Nehmen Sie es!

## 5. Beispiele aus dem Alltag

- *Dimmi tutto!* — Erzähl mir alles!
- *Scusa!* / *Scusi!* — Entschuldige! / Entschuldigen Sie!
- *Senti!* / *Senta!* — Hör mal! / Hören Sie mal!
- *Guarda!* — Schau!
- *Non preoccuparti!* — Mach dir keine Sorgen!

## 6. Häufige Fehler

- *Parli!* (zu Freund) ❌ → *Parla!* ✓
- *Non parla!* (zu Freund) ❌ → *Non parlare!* ✓ (NON + Infinitiv)
- *Lei mi dia* in Anhängung *Dialemi* ❌ → bei Lei: Pronomen vor das Verb
- *Da-mi* getrennt ❌ → *dammi* (zusammengeschrieben mit Verdoppelung)
