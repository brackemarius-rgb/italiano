# Genus (Geschlecht) der Substantive

Substantive sind im Italienischen entweder **maskulin** (männlich) oder **feminin** (weiblich), ein Neutrum gibt es nicht.

## 1. Hauptregel

| Endung | Genus | Beispiele |
|--------|-------|-----------|
| **-o** | meist **maskulin** | *il libro* (Buch), *il cappello* (Hut) |
| **-a** | meist **feminin** | *la casa* (Haus), *la macchina* (Auto) |
| **-e** | maskulin **oder** feminin (lernen!) | *la chiave* (Schlüssel), *il giornale* (Zeitung) |

## 2. Wichtige Ausnahmen

### Weibliche Substantive auf -o

| Italienisch | Deutsch |
|-------------|---------|
| la foto | das Foto |
| la mano | die Hand |
| l'auto | das Auto |
| la radio | das Radio |

### Männliche Substantive auf -a

| Italienisch | Deutsch |
|-------------|---------|
| il cinema | das Kino |
| il problema | das Problem |
| il programma | das Programm |
| il telegramma | das Telegramm |
| il clima | das Klima |
| il pianista | der Pianist |
| il giornalista | der Journalist |

### Substantive auf -i (meist feminin)

| Italienisch | Deutsch |
|-------------|---------|
| la crisi | die Krise |
| la tesi | die These |

### Fremdwörter und auf Konsonant endende Substantive

Sind in der Regel **maskulin**:

- il bar (die Bar)
- il film (der Film)
- lo sport (der Sport)
- l'autobus (der Bus)

## 3. Genus bei Lebewesen

Bei Substantiven, die Lebewesen bezeichnen, stimmt das grammatische Geschlecht meist mit dem natürlichen überein:

| Maskulin | Feminin |
|----------|---------|
| il cugino *(Cousin)* | la cugina *(Cousine)* |
| il bambino *(Junge)* | la bambina *(Mädchen)* |
| il gatto *(Kater)* | la gatta *(Katze)* |

### Bezeichnungen auf -ante, -ente, -ese, -ista

Haben für beide Geschlechter nur **eine Form**:

- il/la giornalista *(Journalist/in)*
- il/la farmacista *(Apotheker/in)*
- il/la cantante *(Sänger/in)*

### Personenbezeichnungen auf -e → weibliche Form auf -essa

- lo studente → la studentessa
- il professore → la professoressa

### Bezeichnungen auf -tore → weibliche Form auf -trice

- l'attore → l'attrice (Schauspieler/in)
- l'autore → l'autrice (Autor/in)

### Völlig verschiedene Formen

| Maskulin | Feminin |
|----------|---------|
| il padre *(Vater)* | la madre *(Mutter)* |
| il fratello *(Bruder)* | la sorella *(Schwester)* |
| l'uomo *(Mann)* | la donna *(Frau)* |

## Häufige Fehler

- *"il foto"* ❌ → *la foto* ✓
- *"la problema"* ❌ → *il problema* ✓
- *"la cinema"* ❌ → *il cinema* ✓
- *"il mano"* ❌ → *la mano* ✓
