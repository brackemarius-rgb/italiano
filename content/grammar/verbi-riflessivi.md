# Reflexive Verben (verbi riflessivi)

Reflexive Verben werden mit einem **Reflexivpronomen** gebildet. Viele Alltagsverben (sich waschen, sich anziehen, aufstehen) sind im Italienischen reflexiv.

## 1. Reflexivpronomen

|             | Pronomen  |
| ----------- | --------- |
| io          | mi        |
| tu          | ti        |
| lui/lei/Lei | si        |
| noi         | ci        |
| voi         | vi        |
| loro        | si        |

## 2. Konjugation – Beispiel: alzarsi (aufstehen)

|             |              |
| ----------- | ------------ |
| io          | mi alzo      |
| tu          | ti alzi      |
| lui/lei     | si alza      |
| noi         | ci alziamo   |
| voi         | vi alzate    |
| loro        | si alzano    |

Das Reflexivpronomen steht **vor dem konjugierten Verb**.

## 3. Wichtige reflexive Verben

| Verb          | Bedeutung           |
| ------------- | ------------------- |
| chiamarsi     | heißen              |
| alzarsi       | aufstehen           |
| svegliarsi    | aufwachen           |
| lavarsi       | sich waschen        |
| vestirsi      | sich anziehen       |
| pettinarsi    | sich kämmen         |
| addormentarsi | einschlafen         |
| riposarsi     | sich ausruhen       |
| divertirsi    | sich amüsieren      |
| annoiarsi     | sich langweilen     |
| sposarsi      | heiraten            |
| trovarsi      | sich befinden       |
| sentirsi      | sich fühlen         |
| arrabbiarsi   | sich aufregen       |
| fermarsi      | anhalten            |

## 4. Im passato prossimo

Reflexive Verben bilden das passato prossimo **immer mit essere** und das Partizip passt sich an:

|             |                       |
| ----------- | --------------------- |
| io          | mi sono alzat**o/a**  |
| tu          | ti sei alzat**o/a**   |
| lui         | si è alzat**o**       |
| lei         | si è alzat**a**       |
| noi         | ci siamo alzat**i/e** |
| voi         | vi siete alzat**i/e** |
| loro (m)    | si sono alzat**i**    |
| loro (f)    | si sono alzat**e**    |

- *Mi sono alzato alle sette.*
- *Maria si è divertita molto.*
- *Ci siamo conosciuti a Roma.*

## 5. Mit Modalverb (volere, potere, dovere)

Das Reflexivpronomen kann **vor dem Modalverb** oder **am Infinitiv** stehen:

- *Mi voglio alzare presto.* = *Voglio alzarmi presto.*
- *Ti devi riposare.* = *Devi riposarti.*

## 6. Beispiele aus dem Tagesablauf

- *Mi sveglio alle sei e mezza.*
- *Mi alzo subito.*
- *Mi lavo e mi vesto.*
- *La sera mi rilasso davanti alla TV.*
- *Mi addormento verso le undici.*

## 7. Häufige Fehler

- *Alzo alle sette* ❌ → *Mi alzo alle sette* ✓
- *Ho mi alzato* ❌ → *Mi sono alzato* ✓ (immer essere!)
- *Maria si è alzato* ❌ → *Maria si è alzata* ✓ (Partizip f)
- *Ci siamo conosciute* nur, wenn beide weiblich sind. Gemischt → *conosciuti*.
