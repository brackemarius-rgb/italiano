# Infinitivsätze: Verb + (Präposition) + Infinitiv

Manche Verben werden **direkt** mit Infinitiv verbunden, andere brauchen die Präposition **a** oder **di**. Das muss man lernen.

## 1. Verb + Infinitiv (ohne Präposition)

Vor allem **Modalverben** und einige weitere:

- volere, potere, dovere, sapere
- preferire, desiderare
- piacere
- fare, lasciare
- vedere, sentire

Beispiele:
- *Voglio andare a casa.*
- *Sa parlare francese.*
- *Mi piace leggere.*
- *Preferisco stare a casa.*

## 2. Verb + **a** + Infinitiv

Verben der **Bewegung, des Anfangens, des Lernens, des Versuchens**:

| Verb       | Beispiel                            |
| ---------- | ----------------------------------- |
| andare a   | *Vado a mangiare.*                  |
| venire a   | *Viene a trovarmi.*                 |
| cominciare a | *Comincio a capire.*              |
| iniziare a | *Inizia a piovere.*                 |
| imparare a | *Imparo a guidare.*                 |
| insegnare a | *Mi insegna a cucinare.*           |
| provare a  | *Provo a chiamare.*                 |
| aiutare a  | *Mi aiuta a studiare.*              |
| riuscire a | *Non riesco a dormire.*             |
| continuare a | *Continuo a lavorare.*            |

## 3. Verb + **di** + Infinitiv

Verben des **Sagens, Denkens, Aufhörens, Versprechens**:

| Verb        | Beispiel                            |
| ----------- | ----------------------------------- |
| finire di   | *Finisco di mangiare.*              |
| smettere di | *Smetto di fumare.*                 |
| pensare di  | *Penso di partire.*                 |
| credere di  | *Credo di sì.*                      |
| decidere di | *Ho deciso di andare.*              |
| dire di     | *Mi ha detto di aspettare.*         |
| chiedere di | *Mi ha chiesto di venire.*          |
| sperare di  | *Spero di vederti.*                 |
| promettere di | *Prometto di tornare.*            |
| cercare di  | *Cerco di capire.*                  |
| ricordarsi di | *Ricordati di chiamare.*          |
| dimenticare di | *Ho dimenticato di chiamare.*    |
| avere voglia di | *Ho voglia di mangiare.*        |
| avere bisogno di | *Ho bisogno di riposare.*      |
| avere paura di | *Ho paura di volare.*            |

## 4. Schnellübersicht häufiger Verben

| ohne | + a       | + di       |
| ---- | --------- | ---------- |
| volere | andare  | finire     |
| potere | venire  | smettere   |
| dovere | cominciare | pensare |
| sapere | imparare | decidere  |
| preferire | provare | dire     |
| piacere | aiutare | sperare    |
| fare    | riuscire | cercare   |

## 5. Beispiele zum Vergleich

- *Voglio mangiare.* (ohne)
- *Comincio **a** mangiare.* (a)
- *Finisco **di** mangiare.* (di)

## 6. Häufige Fehler

- *Comincio mangiare* ❌ → *Comincio a mangiare* ✓
- *Finisco a mangiare* ❌ → *Finisco di mangiare* ✓
- *Ho voglia mangiare* ❌ → *Ho voglia di mangiare* ✓
- *Imparo guidare* ❌ → *Imparo a guidare* ✓

**Tipp:** Diese Verben am besten mit ihrer Präposition zusammen lernen, also nicht nur *cominciare*, sondern *cominciare a*.
