# Plural – Sonderfälle und Unregelmäßigkeiten

## 1. Substantive auf -ca und -ga

→ Plural auf **-che** und **-ghe** (h erhalten, damit harter k/g-Klang bleibt):

| Singular | Plural |
|----------|--------|
| l'amica | le amiche |
| la banca | le banche |
| la riga | le righe |

## 2. Substantive auf -co und -go (vorletzte Silbe betont)

→ Plural auf **-chi** und **-ghi**:

| Singular | Plural |
|----------|--------|
| il lago | i laghi |
| l'albergo | gli alberghi |
| il fungo | i funghi |

**Ausnahme:**

- l'amico → **gli amici** (Plural mit weichem -tschi-Klang)

## 3. Substantive auf -co und -go (drittletzte Silbe betont)

→ Plural auf **-ci** und **-gi**:

| Singular | Plural |
|----------|--------|
| il medico | i medici |
| l'austriaco | gli austriaci |
| lo psicologo | gli psicologi |
| l'asparago | gli asparagi |

## 4. Substantive auf -cia und -gia

- **-cia/-gia mit Konsonant davor** → -ce/-ge (-i fällt weg):
  - l'arancia → **le arance**
  - la camicia → **le camicie**

- **-cia/-gia mit Vokal davor** → -cie/-gie (-i bleibt):
  - la camicia → le camicie

## 5. Substantive auf -io

- Wenn **-i unbetont**: nur ein -i im Plural
  - il negozio → **i negozi**

- Wenn **-i betont**: bleibt erhalten
  - lo z**ì**o → **gli z**ì**i**

## 6. Unregelmäßige Pluralbildungen (auswendig lernen!)

| Singular | Plural |
|----------|--------|
| l'uovo *(das Ei)* | le uova *(die Eier)* — **Genus wechselt!** |
| l'uomo *(der Mann)* | gli uomini |
| il ginocchio *(das Knie)* | le ginocchia *(Genus wechselt)* |
| il braccio *(der Arm)* | le braccia *(Genus wechselt)* |
| l'orecchio *(das Ohr)* | le orecchie *(Genus wechselt)* |
| il dito *(der Finger)* | le dita *(Genus wechselt)* |
| il labbro *(die Lippe)* | le labbra *(Genus wechselt)* |
| la mano *(die Hand)* | le mani |

## 7. Substantive nur im Plural

- la gente *(die Leute)* — **nur Singular!**
- i soldi *(das Geld)* — nur Plural
- gli occhiali *(die Brille)* — nur Plural
- i pantaloni *(die Hose)* — nur Plural
- gli spiccioli *(das Kleingeld)* — nur Plural
- gli spinaci *(der Spinat)* — nur Plural
- i baffi *(der Schnurrbart)* — nur Plural

## Merksatz für die Körperteile

Viele Körperteile, die paarweise vorkommen, wechseln im Plural das Genus von maskulin (-o) zu feminin (-a):

- *il braccio → le braccia*
- *il ginocchio → le ginocchia*
- *l'orecchio → le orecchie*
- *il dito → le dita*

## Häufige Fehler

- *"i amichi"* ❌ → *gli amici* ✓
- *"i uovi"* ❌ → *le uova* ✓
- *"le mane"* ❌ → *le mani* ✓
- *"le bracci"* ❌ → *le braccia* ✓
