# Präsens – wichtige unregelmäßige Verben

Viele der häufigsten Verben sind unregelmäßig. Die wichtigsten musst du auswendig lernen.

## 1. Bewegung & Sein

|             | andare (gehen) | venire (kommen) | stare (sich befinden) |
| ----------- | -------------- | --------------- | --------------------- |
| io          | vado           | vengo           | sto                   |
| tu          | vai            | vieni           | stai                  |
| lui/lei     | va             | viene           | sta                   |
| noi         | andiamo        | veniamo         | stiamo                |
| voi         | andate         | venite          | state                 |
| loro        | vanno          | vengono         | stanno                |

## 2. Tun, geben, sagen

|             | fare (machen) | dare (geben) | dire (sagen) |
| ----------- | ------------- | ------------ | ------------ |
| io          | faccio        | do           | dico         |
| tu          | fai           | dai          | dici         |
| lui/lei     | fa            | dà           | dice         |
| noi         | facciamo      | diamo        | diciamo      |
| voi         | fate          | date         | dite         |
| loro        | fanno         | danno        | dicono       |

## 3. Wollen, können, müssen, wissen, mögen (Modalverben)

|             | volere (wollen) | potere (können) | dovere (müssen) | sapere (wissen) | piacere |
| ----------- | --------------- | --------------- | --------------- | --------------- | ------- |
| io          | voglio          | posso           | devo            | so              | piaccio |
| tu          | vuoi            | puoi            | devi            | sai             | piaci   |
| lui/lei     | vuole           | può             | deve            | sa              | piace   |
| noi         | vogliamo        | possiamo        | dobbiamo        | sappiamo        | piacciamo |
| voi         | volete          | potete          | dovete          | sapete          | piacete |
| loro        | vogliono        | possono         | devono          | sanno           | piacciono |

## 4. Weitere wichtige

|             | uscire | bere    | tenere  | salire  | scegliere |
| ----------- | ------ | ------- | ------- | ------- | --------- |
| io          | esco   | bevo    | tengo   | salgo   | scelgo    |
| tu          | esci   | bevi    | tieni   | sali    | scegli    |
| lui/lei     | esce   | beve    | tiene   | sale    | sceglie   |
| noi         | usciamo| beviamo | teniamo | saliamo | scegliamo |
| voi         | uscite | bevete  | tenete  | salite  | scegliete |
| loro        | escono | bevono  | tengono | salgono | scelgono  |

## 5. Beispielsätze

- *Vado al cinema.* — Ich gehe ins Kino.
- *Cosa fai stasera?* — Was machst du heute Abend?
- *Voglio un caffè.* — Ich möchte einen Kaffee.
- *Esco con gli amici.* — Ich gehe mit Freunden aus.
- *Bevo solo acqua.* — Ich trinke nur Wasser.

## 6. Häufige Fehler

- *ando al cinema* ❌ → *vado al cinema* ✓
- *facio* ❌ → *faccio* ✓
- *poto* ❌ → *posso* ✓
- *uscio* ❌ → *esco* ✓ (Stammwechlsel u→e)
- *bevono* ist die korrekte 3. Pl. (nicht *berono* oder *beono*)
