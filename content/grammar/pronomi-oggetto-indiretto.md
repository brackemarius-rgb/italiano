# Indirekte Objektpronomen (Dativ)

Indirekte Objektpronomen antworten auf die Frage **"Wem?"** Sie stehen normalerweise **vor dem Verb**.

## Die Formen

| Person | Pronomen | Bedeutung |
|--------|----------|-----------|
| 1. Sg. | **mi** | mir |
| 2. Sg. | **ti** | dir |
| 3. Sg. m. | **gli** | ihm |
| 3. Sg. f. | **le** | ihr |
| 3. Sg. Höflichkeit | **Le** | Ihnen *(großgeschrieben)* |
| 1. Pl. | **ci** | uns |
| 2. Pl. | **vi** | euch |
| 3. Pl. m./f. | **gli** *(früher: loro)* | ihnen |

## Vergleich direkt vs. indirekt

Aufgepasst — viele Formen sind gleich!

| | Direkt (Akk.) | Indirekt (Dat.) |
|---|---------------|-----------------|
| 1. Sg. | mi | mi |
| 2. Sg. | ti | ti |
| 3. Sg. m. | **lo** | **gli** |
| 3. Sg. f. | **la** | **le** |
| 1. Pl. | ci | ci |
| 2. Pl. | vi | vi |
| 3. Pl. m. | **li** | **gli** |
| 3. Pl. f. | **le** | **gli** |

**Merke:** lo/la → ihn/sie | gli/le → ihm/ihr

## Beispiele

- *Telefono a Maria → **Le** telefono.* *(Ich rufe sie an / ihr rufe ich an.)*
- *Scrivo a Marco → **Gli** scrivo.* *(Ich schreibe ihm.)*
- *Regalo un libro a mio fratello → **Gli** regalo un libro.*

## Verben mit indirektem Objekt

Wichtige Verben, die ein indirektes Objekt verlangen:
- **dare a** *(geben)* → Le do un libro.
- **dire a** *(sagen)* → Gli dico la verità.
- **chiedere a** *(fragen)* → Le chiedo un favore.
- **rispondere a** *(antworten)* → Gli rispondo.
- **scrivere a** *(schreiben)* → Le scrivo una lettera.
- **telefonare a** *(anrufen)* → Gli telefono.
- **regalare a** *(schenken)* → Le regalo dei fiori.
- **piacere a** *(gefallen)* → Mi piace il caffè.

## Spezialfall: piacere

Das Verb **piacere** (gefallen) wird in fast allen Fällen mit indirektem Objektpronomen gebildet:

- *Ti piace la pizza?* *(Gefällt **dir** die Pizza?)*
- *Mi piacciono gli spaghetti.* *(**Mir** gefallen die Spaghetti.)*
- *Le piace la musica?* *(Gefällt **ihr/Ihnen** die Musik?)*

Anpassung: *piace* (Singular) / *piacciono* (Plural) — passt sich an, was gefällt.

## Stellung im Satz

Wie direkte Pronomen — **vor dem konjugierten Verb**:

- *Le telefono.* *(Ich rufe sie/Sie an.)*
- *Le posso telefonare.* / *Posso telefonar**le**.*

Bei zusammengesetzten Verben (Modalverb + Infinitiv) sind beide Stellungen möglich.

## Häufige Fehler

- *"Lo telefono"* ❌ → *Gli telefono* ✓ (telefonare a → indirekt)
- *"Le aspetto"* ❌ (für "ich warte auf sie") → *La aspetto* ✓ (aspettare → direkt)
- *"Mi piaccio gli spaghetti"* ❌ → *Mi piacciono...* ✓
- *"A me piace il caffè"* ist auch richtig (betont)
