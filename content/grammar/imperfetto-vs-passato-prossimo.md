# Imperfetto vs. passato prossimo

Beide Zeiten beziehen sich auf Vergangenheit, aber haben **unterschiedliche Funktionen**. In der Erzählung werden sie oft kombiniert.

## 1. Faustregel

| Imperfetto                                | Passato prossimo                       |
| ----------------------------------------- | -------------------------------------- |
| Zustand, Beschreibung, Gewohnheit         | Einzelne, abgeschlossene Handlung     |
| Hintergrund einer Erzählung               | Vordergrund: Was ist passiert?        |
| **Wie war es?**                           | **Was ist passiert?**                 |
| dauerhaft, wiederholt, parallel           | punktuell, einmalig                   |

## 2. Beispiele zum Vergleich

| Imperfetto                              | Passato prossimo                          |
| --------------------------------------- | ----------------------------------------- |
| *Da bambino mangiavo molti dolci.*      | *Ieri ho mangiato un dolce.*              |
| *Ogni estate andavo al mare.*           | *L'estate scorsa sono andato in Sicilia.* |
| *Il sole splendeva.*                    | *Il sole è uscito alle 7.*                |
| *Avevo paura.*                          | *Ho avuto paura quando l'ho visto.*       |

## 3. Klassische Erzählstruktur

Hintergrund (imperfetto) + Ereignis (passato prossimo):

> *Era una sera fredda. Pioveva e tirava vento. Io leggevo un libro vicino al camino. **All'improvviso è suonato il telefono.** Mi sono alzato e ho risposto.*

- *Era / pioveva / tirava / leggevo* → Hintergrund
- *è suonato / mi sono alzato / ho risposto* → Ereignisse

## 4. Signalwörter

**Imperfetto** → sempre, spesso, di solito, ogni giorno, mentre, da bambino, tutti i giorni

**Passato prossimo** → ieri, una volta, all'improvviso, l'altro ieri, l'anno scorso, due ore fa

## 5. Bedeutungsunterschiede bei manchen Verben

| Verb     | Imperfetto                            | Passato prossimo                      |
| -------- | ------------------------------------- | ------------------------------------- |
| sapere   | *Sapevo che…* (ich wusste)            | *Ho saputo che…* (ich habe erfahren)  |
| conoscere| *Conoscevo Marco.* (kannte)           | *Ho conosciuto Marco.* (lernte kennen) |
| volere   | *Volevo andare.* (wollte – ohne Folge)| *Ho voluto andare.* (habe gewollt – tat es) |
| dovere   | *Dovevo studiare.* (sollte/musste)    | *Ho dovuto studiare.* (musste tatsächlich) |

## 6. Häufige Fehler

- *Ieri faceva caldo e sono andato al mare.* ✓ (faceva = Beschreibung, sono andato = Ereignis)
- *Quando ero piccolo ho giocato sempre a calcio.* ❌ → *…giocavo sempre…* (Gewohnheit!)
- *La casa è stata grande.* ❌ → *La casa era grande.* (Beschreibung!)
