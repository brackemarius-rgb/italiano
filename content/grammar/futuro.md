# Futuro semplice

Das Futuro semplice drückt zukünftige Handlungen, Vorhersagen und Vermutungen aus. In der gesprochenen Sprache wird es oft durch das Präsens ersetzt.

## 1. Bildung – regelmäßig

Endung **-ò, -ai, -à, -emo, -ete, -anno**, an den Stamm gehängt:

|             | parlare    | prendere   | dormire   |
| ----------- | ---------- | ---------- | --------- |
| io          | parlerò    | prenderò   | dormirò   |
| tu          | parlerai   | prenderai  | dormirai  |
| lui/lei     | parlerà    | prenderà   | dormirà   |
| noi         | parleremo  | prenderemo | dormiremo |
| voi         | parlerete  | prenderete | dormirete |
| loro        | parleranno | prenderanno| dormiranno|

**Achtung:** Bei -are-Verben wird das **a** zum **e** (parlare → parler-).

## 2. Wichtige unregelmäßige Stämme

| Infinitiv | Stamm  | 1. Pers.    |
| --------- | ------ | ----------- |
| essere    | sar-   | sarò        |
| avere     | avr-   | avrò        |
| andare    | andr-  | andrò       |
| venire    | verr-  | verrò       |
| fare      | far-   | farò        |
| dare      | dar-   | darò        |
| vedere    | vedr-  | vedrò       |
| sapere    | sapr-  | saprò       |
| potere    | potr-  | potrò       |
| volere    | vorr-  | vorrò       |
| dovere    | dovr-  | dovrò       |
| stare     | star-  | starò       |
| rimanere  | rimarr-| rimarrò     |
| tenere    | terr-  | terrò       |

## 3. Verwendung

### a) Zukunft

- *Domani partirò per Roma.* — Morgen werde ich nach Rom abreisen.
- *L'anno prossimo cambieranno casa.* — Nächstes Jahr werden sie umziehen.

### b) Vermutung in der Gegenwart (!)

- *Saranno le tre.* — Es wird (jetzt) drei Uhr sein.
- *Avrà 40 anni.* — Er/sie wird (wohl) 40 sein.
- *Dove sarà Marco?* — Wo Marco wohl ist?

### c) Versprechen, Pläne

- *Ti chiamerò.* — Ich werde dich anrufen.

## 4. Tipp

In der **gesprochenen Sprache** wird statt Futuro oft das **Präsens** + Zeitangabe verwendet:
- *Domani parto per Roma.* (statt *partirò*)
- *Stasera mangiamo a casa.* (statt *mangeremo*)

Das Futuro wird häufiger benutzt, wenn:
- Vermutungen ausgedrückt werden
- die Handlung weiter weg in der Zukunft liegt
- sich Förmlichkeit ausdrücken soll

## 5. Häufige Fehler

- *parlero* ❌ → *parlerò* ✓ (Akzent auf der letzten Silbe)
- *andarò* ❌ → *andrò* ✓ (unregelmäßiger Stamm)
- *avero* ❌ → *avrò* ✓
- *vorrò* (volere) und *verrò* (venire) verwechselt
