# Der Superlativ

Es gibt zwei Arten von Superlativ im Italienischen:

## 1. Relativer Superlativ (mit Vergleich)

= "der/die/das ... -ste" — Vergleich mit einer Gruppe.

**Bildung:** bestimmter Artikel + più/meno + Adjektiv

| Form | Beispiel |
|------|----------|
| il/la più ... | la **più bella** regione *(die schönste Region)* |
| il/la meno ... | il **meno caro** ristorante *(das günstigste Restaurant)* |

Beispiele:
- *La Toscana è **la regione più bella** d'Italia.*
- *Marco è **il più simpatico** dei miei amici.*

## 2. Absoluter Superlativ (ohne Vergleich)

= "sehr ..." — Steigerung ohne Vergleich.

**Bildung:** Adjektiv-Stamm + **-issimo / -issima / -issimi / -issime**

| Adjektiv | Absoluter Superlativ |
|----------|----------------------|
| buono | buon**issimo** |
| bello | bell**issimo** |
| dolce | dolc**issimo** |
| veloce | veloc**issimo** |
| fresco | fresch**issimo** *(h, um harten k-Klang zu erhalten)* |
| stanco | stanch**issimo** |

Beispiele:
- *Il caffè è **buonissimo**.* *(Der Kaffee ist sehr gut.)*
- *Le pere sono **dolcissime**.* *(Die Birnen sind sehr süß.)*
- *Sto **benissimo**.* *(Mir geht es sehr gut.)*

## 3. Alternative für absoluten Superlativ

Statt -issimo kann man auch verwenden:
- **molto** + Adjektiv → *molto buono*
- **tanto** + Adjektiv → *tanto bello*
- **veramente** + Adjektiv → *veramente bello*

## 4. Unregelmäßige absolute Superlative

Einige Adjektive haben auch eine alte/lateinische Form:

| Adjektiv | Regulär | Unregelmäßig (selten) |
|----------|---------|------------------------|
| buono | buonissimo | ottimo |
| cattivo | cattivissimo | pessimo |
| grande | grandissimo | massimo |
| piccolo | piccolissimo | minimo |

Beispiel:
- *La qualità è **ottima**.* *(Die Qualität ist sehr gut.)*

## Häufige Fehler

- *"È più bella regione"* ❌ → *È **la** più bella regione* ✓ (Artikel!)
- *"frescissimo"* ❌ → *fresch**issimo*** ✓ (h vor -issimo)
- *"buonissimo"* ✓
- *"molto buonissimo"* ❌ (nicht beides) → *molto buono* oder *buonissimo* ✓
