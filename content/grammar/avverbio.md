# Das Adverb

Adverbien beschreiben Verben, Adjektive oder andere Adverbien. Sie sind **unveränderlich** (nie Endungen anpassen).

## 1. Bildung mit -mente

Aus den meisten Adjektiven bildet man Adverbien mit der Endung **-mente** (entspricht etwa dem deutschen *-weise* oder einem Adverb).

### Adjektive auf -o → feminine Form + -mente

| Adjektiv | Adverb |
|----------|--------|
| lento (langsam) | lenta**mente** |
| veloce (schnell) | veloce**mente** |
| profondo (tief) | profonda**mente** |
| vero (wahr) | vera**mente** *(wirklich)* |
| esatto (genau) | esatta**mente** |
| improvviso (plötzlich) | improvvisa**mente** |
| probabile | probabil**mente** |

### Sonderfall: Adjektive auf -le, -re

Bei Adjektiven auf **-le** und **-re** entfällt das **-e** vor -mente:

| Adjektiv | Adverb |
|----------|--------|
| facile | faci**l-mente** |
| difficile | difficil**mente** |
| regolare | regolar**mente** |
| naturale | natural**mente** |
| finale | final**mente** |

## 2. Häufige unregelmäßige Adverbien

Diese muss man auswendig lernen:

| Adverb | Bedeutung |
|--------|-----------|
| **bene** | gut |
| **male** | schlecht |
| **molto** | viel, sehr |
| **poco** | wenig |
| **tanto** | viel, so sehr |
| **troppo** | zu viel/sehr |
| **abbastanza** | genug, ziemlich |
| **sempre** | immer |
| **spesso** | oft |
| **mai** | nie |
| **già** | schon |
| **non ... ancora** | noch nicht |
| **subito** | sofort |
| **adesso/ora** | jetzt |
| **presto** | früh, bald |
| **tardi** | spät |
| **anche** | auch |

## 3. Wichtig: Adjektiv vs. Adverb

| Adjektiv | Adverb |
|----------|--------|
| Lui è **buono**. *(Er ist gut.)* | Lui parla **bene**. *(Er spricht gut.)* |
| Il caffè è **cattivo**. | Sto **male**. |
| Lui è **veloce**. | Lui corre **velocemente**. |

## 4. Stellung des Adverbs

- Bei Verben im einfachen Tempus: **nach** dem Verb
  - *Parla **bene** italiano.*
- Bei Verben im zusammengesetzten Tempus: meist zwischen Hilfsverb und Partizip
  - *Ho **già** mangiato.*
- *Sempre, mai, già, ancora* stehen bei zusammengesetzten Tempora **zwischen Hilfsverb und Partizip**:
  - *Non ho **mai** visto un film così.*

## Häufige Fehler

- *"Lui parla buono"* ❌ → *Lui parla bene* ✓
- *"facilmente"* (vom Adj. facile) ❌ → *facilmente* ✓ (richtig, nur ein 'l')
- *"Ho mangiato già"* ❌ → *Ho **già** mangiato* ✓
